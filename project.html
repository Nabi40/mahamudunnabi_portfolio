jersey0099

Key Features: Rather than learning jersey number recognition as a set of fixed classes, this model breaks down the prediction of jersey numbers into two separate predictions for the tens digit and ones digit. By doing so, this model allows for each digit from 0 to 9 to be learned as a reusable visual pattern rather than learning each full number pattern. Thus, this model is able to produce a correct prediction for any combination of jersey numbers that may never have been seen before by combining its predictions. A model that was trained with “4,” “9,” and “49” would still be able to produce a correct prediction for “94,” “23,” and “77.”

Tech: MobileNetV3-Small, pytorch, GRU
